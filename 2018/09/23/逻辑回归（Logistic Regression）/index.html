逻辑回归（Logistic Regression）

发表于 2018-09-23 | 分类于 基础算法

逻辑回归

逻辑回归VS线性回归

逻辑回归不是回归（线性回归是利用数理统计中回归分析，来确定两种或两种以上变量间相互依赖的定量关系的一种统计分析方法），而是二分类器，分类结果对影响因素之间的一种多变量分析方法，判断多个变量对于结果的影响。

$x_1,x_2…x_k$是每条数据的$k$维特征。
直接将$x$作为模型就是线性回归了，但是引入了一个如下公式的函数，该函数也叫激励函数，将线性的函数转换为非线性，实现特征到分类结果的映射，实现逻辑回归。线性回归的直线你和所有数据的点，逻辑回归则用直线作为不同类别的分界线。逻辑回归也可以理解成被激励函数归一化后的线性回归。

代价函数
